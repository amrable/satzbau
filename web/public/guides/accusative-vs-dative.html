Accusative vs dative in German

The core idea, the article endings, the verbs that break the pattern, and worked examples.

The core idea in one line

Accusative = the direct object (the thing being acted on). Dative = the indirect object (the recipient / the one affected).

English uses word order for this. German uses case. Compare:

  • I give the man the book. → Ich gebe dem Mann das Buch.
  • dem Mann = dative (the recipient). das Buch = accusative (the thing given).

Ask two questions:

  1. Wen/was? (Whom/what [is acted on]?) → accusative.
  2. Wem? (To/for whom?) → dative.

Notice the shortcut: only the masculine noun changes in accusative (der → den). The dative changes everything. That's why masculine nouns are where most learners get stuck first.

Indefinite articles follow the same pattern: ein/eine/ein → einen/eine/ein (acc) and einem/einer/einem (dat).

Verbs that force dative

Some verbs always take a dative object even though English treats them as direct. Memorize these — they're the most common cause of mistakes:

  • helfen — Ich helfe dir. (not dich)
  • danken — Ich danke dem Lehrer.
  • antworten — Sie antwortet mir.
  • folgen — Der Hund folgt dem Mann.
  • gefallen — Das Buch gefällt mir.
  • gehören — Das Auto gehört meinem Bruder.
  • glauben (to a person) — Ich glaube dir.
  • passen, schmecken, fehlen, begegnen

Prepositions fix the case

Some prepositions always take accusative (durch, für, gegen, ohne, um), some always dative (aus, bei, mit, nach, seit, von, zu), and the two-way prepositions (in, an, auf, über, unter, vor, hinter, neben, zwischen) switch based on motion vs location. See the two-way prepositions guide.

Motion vs location test

  • Ich gehe in die Schule. (motion → accusative)
  • Ich bin in der Schule. (location → dative)

Worked examples

Er gibt seiner Schwester einen Brief. — seiner Schwester dative (recipient), einen Brief accusative (object given).

Sie hilft dem alten Mann. — dative because helfen is a dative verb.

Wir legen das Buch auf den Tisch. — accusative because there's motion onto the table.

Das Buch liegt auf dem Tisch. — dative because it's a static location.
